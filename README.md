# Medical Journal of International Multidisciplinary Research (MJIMR)

Official website for MJIMR - An international peer-reviewed open access medical journal.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Project Structure

```
mjimr/
├── app/                    # Next.js App Router pages
├── components/             # React components
├── data/                   # Journal data and configuration
├── public/                 # Static files
└── lib/                    # Utility functions
```

## Features

- Current issue display
- Article archives
- Editorial board
- Author guidelines
- Contact information
- Responsive design
- SEO optimized

## Deployment

Deploy to Vercel:
```bash
npm run build
```
